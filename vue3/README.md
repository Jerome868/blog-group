## 安装依赖
```bash
# 安装 vue3.0
yarn add vue@next

# 安装 vue 相关解析依赖
yarn add vue-loader@next @vue/compiler-sfc -D

# 安装 webpack 相关
yarn add webpack webpack-cli webpack-dev-server -D

# 安装打包插件
yarn add html-webpack-plugin clean-webpack-plugin -D

# 安装样式编辑依赖
yarn add style-loader css-loader less less-loader -D
```
